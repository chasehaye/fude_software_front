## Prerequisites
Node version -> 24.14.1
Npm version -> 11.11.0

## To get started locally:
First get your own local copy of the code via:
`git clone https://github.com/chasehaye/fude_software_front.git`
`cd fude_software_front`

Then run:
`npm install`

Then checkout to a branch for your own work - This keeps the main branch clean and makes the Pull Request process much smoother:
`git checkout -b feat/your-feature-name`

Run the development server:
`npm run dev`

## Pretteir
(Make sure to format code correctly before submission for pull requests)
To check for TypeScript errors: 
`npm run format`

To fix everything for format: 
`npm run check`

## Verify buld 
Verify that the project builds correctly
`npm run build`

## To submit a pull request
After making the development changes you would like seen in the pull request:
`git add .`
`git commit -m "feat: added new profile dashboard"`
`git push origin feat/your-feature-name`
